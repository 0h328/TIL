1. 전위 순회: 부모노드 방문 후, 좌식노드를 좌, 우 순서로 방문
A -> B -> D -> H -> I -> E -> J -> C -> F -> K -> G -> L -> M

2. 중위 순회: 왼쪽 자식노드, 부모노드, 오른쪽 자식노드 순으로 방문
H -> D -> I -> B -> J -> E -> A -> F -> K -> C -> L -> G -> M

3. 후위 순회: 자식노드를 좌우 순서로 방문한 후, 부모노드로 방문
H -> I -> D -> J -> E -> B -> K -> F -> L -> M -> G -> C -> A
